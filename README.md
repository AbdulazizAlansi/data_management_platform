# DataHarbor
![Data Harbor](https://github.com/AbdulazizAlansi/data_management_platform/blob/main/doc/Data%20Management%20Platform.png)
DataHarbor is a data quality monitoring and analytics platform designed to help organizations analyze datasets, detect data quality issues, and generate insights from structured data.

![About Data Harbor](https://github.com/AbdulazizAlansi/data_management_platform/blob/main/doc/ABOUT%20DATAHARBOR.png)
DataHarbor brings together everything you need to manage, understand, and trust your data — from upload to insight, all in one intuitive workspace.

![Dashboard](https://github.com/AbdulazizAlansi/data_management_platform/blob/main/doc/Dashboard.png)
An interactive dashboard that provides an overview of dataset quality, including missing values, duplicates, and validation results, helping users quickly identify and monitor data issues.

![Weak Quality File](https://github.com/AbdulazizAlansi/data_management_platform/blob/main/doc/Poor%20quality%20file.png)
Indicates that the uploaded dataset contains multiple data quality issues such as missing values, duplicates, or invalid formats.

![Governance Dashboard](https://github.com/AbdulazizAlansi/data_management_platform/blob/main/doc/Governance%20Dashboard.png)
Dashboard that monitors data governance metrics, ensuring datasets follow quality, consistency, and management standards.

## Features

- Dataset upload and analysis
- Data quality validation
- Missing value detection
- Duplicate detection
- Invalid date and numeric validation
- Automated quality reporting
